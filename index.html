📋 Setup Instructions

Step 1: Create Google Sheet

Create a new Google Sheet with these exact column headers in Row 1:

Date | Entry ID | Venmo | Check | Invoice | Zelle | Stripe | Cash | Total Income | Taxable Income | Tax Transfer | Employee Names | Employee Hours | Employee Pay | Total Employee Cost | Hunter Tips | Hunter Cash Draws | Hunter Checks | Hunter Total | Kullen Tips | Kullen Cash Draws | Kullen Checks | Kullen Total | Total Owner Payouts | Food | Supplies | Gas/Auto | Total Expenses | Net Profit | Timestamp

Step 2: Get API Key

  1. Go to Google Cloud Console
  2. Create a new project or select existing
  3. Enable "Google Sheets API"
  4. Go to Credentials → Create Credentials → API Key
  5. Copy the API key
  6. Restrict the key to "Google Sheets API" only (recommended)

Step 3: Share Your Sheet

  1. Open your Google Sheet
  2. Click Share button
  3. Set to "Anyone with the link can edit"
  4. Copy the Spreadsheet ID from the URL (between /d/ and /edit)

Step 4: Enter Configuration

Paste your API Key and Spreadsheet ID in the form above and click "Save Configuration".

Step 5: Test & Start Using

Click "Test Connection" to verify everything works. Then switch to the "Daily Entry" tab to start tracking!
